# Vidi Test Pijar - Backend App

Aplikasi backend Laravel dengan fitur:
- Autentikasi JWT (Register, Login, Logout)
- Manajemen Data Karyawan (CRUD)
- Pagination
- Resource API Response

## Teknologi yang Digunakan
- PHP 8.x
- Laravel 10.x
- JWT Authentication
- Laravel/Sail
- Postgresql
- Docker
